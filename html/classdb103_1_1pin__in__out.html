LPC1114 pin_out constructor.

Constructor for an LPC1114 input pin.

This constructor doesn't set the pin direction to input or output, a direction_set function must be called to do so.

void db103::pin_in_out::flush ( )

flush the pin buffer

This function flushes the pin write buffer: if the buffered value has not yet been written it is written to the pin. When the value has been written, it might be written again.

The pin write buffer might also be fluhsed as a side-effect of other operations.
