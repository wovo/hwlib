Detailed Description

Nokia 5510 B/W graphics LCD library.

This class implements an interface to the type of LCD that was used in older Nokia telephones. It is a 84 columns x 48 lines black-and-white LCD.

This type of LCD is cheap and available from lots of sources, but the quality is often abominable.

clear the window

This function clears the windows by writing the background color to all pixels. The default implementation writes to all pixels in sequence. A concrete window can probably provide a faster implementation.

When buffering is specified ( buf == buffered ) the actual clearing can be delayed until the next flush() call.
